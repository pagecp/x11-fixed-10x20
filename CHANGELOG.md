# Changelog

## 1.0.0 — 2026-07-21

- Recovery of Christian Pagé's exact 2003 `FixedRegular` Mac bitmap SFNT.
- First modern Unicode companion TrueType release.
- 5,204 encoded characters converted from the X.Org 10×20 Unicode PCF.
- Original BDF, ISO-8859, KOI8-R and Unicode PCF files preserved.
- Reproducible FontTools conversion and specimen generation.
- macOS one-click user installation.
