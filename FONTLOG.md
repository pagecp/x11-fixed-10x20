# FONTLOG — X11 Fixed 10×20

## FixedRegular Original (2003)

Converted by Christian Pagé for Mac OS X using FontLab 3. The unmodified SFNT
was recovered in 2026 from resource type `sfnt`, ID 18772, inside the original
Mac FOND suitcase. Its internal metadata includes `FONTLAB30:TTEXPORT`.

- Family: Fixed
- Full name: FixedRegular
- Format: Apple bitmap SFNT (`bdat`/`bloc`)
- Native strike: 20 px
- Glyphs: 223
- Rendering: non-scalable, non-antialiased original bitmap

## 1.0.0 (2026-07-21)

The X.Org Unicode `10x20.pcf.gz` strike was converted to quadratic TrueType
outlines. Each source pixel maps to a 50×50 font-unit square in a 1000 UPM em.
No glyph design was changed.

Naming was changed from the generic historical family name `Misc Fixed` to
`X11 Fixed 10x20` so the installable face can coexist with X11 bitmap font
packages and other sizes in the Misc Fixed family.

Metrics:

- Units per em: 1000
- Advance width: 500 (10 source pixels)
- Ascender: 800 (16 source pixels)
- Descender: -200 (4 source pixels)
- Source strike: 20 ppem
- Encoded characters: 5,204
